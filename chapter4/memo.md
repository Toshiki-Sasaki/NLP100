##### mecab のインストール
```
Mecabと辞書のインストール
$ brew install mecab
$ brew install mecab-ipadic

Mecab-python3 をインストール
$ pip install mecab-python3
```

##### mecab の使い方
```
インスタンス化
$ mecab = MeCab.Tagger('-Ochasen')

parseする
$ mecab.parse(hoge.txt)
```
